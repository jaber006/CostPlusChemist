System Architecture

Six-Layer Architecture

A full-stack system from data ingestion to order fulfillment, with compliance and AI automation built into every layer.

1
Data Ingestion — Sigma Sync
Automated product data extraction from SigmaConnect portal
Python scraper/bot logs into SigmaConnect portal
AI-assisted product data extraction (name, description, images, wholesale price, schedule, stock)
Runs on schedule — daily or twice daily sync
Stores raw data in staging database
AI categorizes and enriches product data
2
Product Database & Pricing Engine
Automatic pricing calculation and catalogue management
PostgreSQL product catalogue database
Automatic price: wholesale_cost × 1.15
Product categorization (General Sale, S2, S3, S4)
Stock availability tracking
Price history tracking
Search index (Meilisearch or Algolia)
3
Web Application — Next.js
Mobile-first frontend with transparent pricing on every product
Next.js with server-side rendering
Product browsing with search + filters
Product pages with transparent price breakdown
S2/S3 compliance workflow (questionnaire → review)
eScript integration (token upload, Active Script List)
User accounts, order history, reorder
"Meet MJ" — book consult, call, email
Mobile-first responsive design
4
Order & Fulfillment
From cart to doorstep — payments, shipping, and subscriptions
Shopping cart + Stripe checkout
Order management dashboard (for MJ)
Australia Post / Sendle shipping integration
Postage calculator (weight × destination)
Order status tracking + notifications
Auto-reorder / subscription option
5
Compliance & Regulatory
TGA, Pharmacy Board, and medicine schedule compliance
S2: online questionnaire before purchase
S3: pharmacist personally approves (MJ reviews + contacts patient)
S4: eScript token verification via NPDS
TGA advertising compliance (no therapeutic claims on S3+)
QCPP accreditation display
Pharmacy Board registration details
Record keeping and audit trail
6
AI & Automation
Smart search, chatbot, and automated workflows
AI product data extraction from Sigma portal
AI-powered search ("something for a headache")
Chatbot for basic pharmacy queries
Automated S2 screening questionnaires
Smart reorder suggestions from purchase history
Tech Stack

Recommended Technologies

A modern, proven stack optimised for Australian conditions — fast search, compliant checkout, reliable shipping APIs.

Frontend
Next.js 14+ (App Router)
Tailwind CSS + shadcn/ui. Server-side rendering for SEO. Mobile-first.
Backend
Next.js API + Python (FastAPI)
API routes for web requests. Python FastAPI for Sigma scraper/sync engine.
Database
PostgreSQL (Supabase)
Product catalogue, orders, users. Supabase for auth, realtime, and row-level security.
Search
Meilisearch
Self-hosted, fast, typo-tolerant. Instant search with fuzzy matching and filters.
Payments
Stripe
PCI-compliant checkout. Subscription support for auto-reorder. Webhooks for order flow.
Shipping
Australia Post + Sendle API
Rate calculation, label generation, tracking. Sendle for eco-friendly options.
Auth
NextAuth.js or Clerk
Email/password + social login. Clerk for faster setup with pre-built UI.
Hosting
Vercel + Railway/Fly.io
Vercel for frontend (edge). Railway or Fly.io for backend services and cron jobs.
AI
OpenAI API
Product enrichment, natural language search, chatbot. GPT-4 for compliance-sensitive tasks.
Monitoring
Sentry + Plausible Analytics
Error tracking + privacy-friendly analytics. No cookies, GDPR compliant.
